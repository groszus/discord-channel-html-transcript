# Contributing

💖 thank you for considering contributing! Any form of contribution means a lot.

If you don't have the time to contribute code, you can still support the project by:

- **Starring** the repository.
- **Sharing** the project with others.

## Guide

### I Have A Question

If you're stuck, need clarification, or just want to discuss the project, the community is here to help!

You can join the [Discord Server](https://discord.omardiaa.dev) to ask us directly.

## Development

### Prerequisites

- **Java 17** or higher.

### Clone The Repository

1. **Clone the repository:**
    ```shell
    git clone https://github.com/omardiaadev/discord-html-transcript.git
    ```

2. **Configure the environment:**
    <table>
        <tr>
            <th>Variable</th>
            <th>Description</th>
        </tr>
        <tr>
            <td><code>JTE_DEV</code></td>
            <td>Allows you to compile templates to Java classes on demand.</td>
        </tr>
    </table>

3. **Run:**\
   If you're using IntelliJ IDEA, see [Run Configurations](.idea/runConfigurations).
