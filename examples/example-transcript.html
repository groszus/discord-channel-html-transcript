    Packages

    1. discord-html-transcript-api: Core package, built with Java.
    2. discord-html-transcript-jda: JDA Wrapper
    3. discord-html-transcript-discordjs: Discord.js Wrapper
    4. discord-html-transcript-discordpy: Discord.py Wrapper

    1. discord-html-transcriber

    Content

    - Includes a JSON schema that's required to generate the HTML using JTE.

    Flow

    Built into a native image using GraalVM that's downloaded by the wrappers.
    When ran, it offers an API endpoint locally to accept a payload (Messages) from the wrapper.

    2. discord-html-transcriber-js

    Content

    - Includes a JSON schema (DTO) to match discord-html-transcriber's schema.

    Flow

    Downloads the discord-html-transcriber respective to the OS, runs it locally.
    Retrieves the Messages using Discord.js, passes it to discord-html-transcriber's local server to generate the HTML.
